# Bookstore Android Mobile Client

An **Android mobile client application** built in **Java** for an online bookstore. The app features a book catalog with genre filtering, shopping cart, and user authentication.

---

## Overview

**STR Bookstore** is an Android application that allows users to browse, search, and purchase books. The app provides a seamless shopping experience with a book catalog, genre-based filtering, shopping cart management, and user authentication.

### Key Use Cases
- Browse book catalog with genre filtering
- Search for books by title
- Add books to shopping cart
- Manage cart items
- Complete purchases with card payment details
- User registration and login
